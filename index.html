🚀 3ステップで簡単スタート！

  2. 🔴 録音開始: 赤いボタンを押して録音開始
  3. 🤖 要約作成: Claude連携で自動要約を作成

📝 使い方ガイド

🎤 電話録音の手順

  1. 📱 スマートフォンでスピーカーフォンをONにする
  2. 🎤 アプリで「録音開始」をタップ
  3. 📞 電話を開始（環境音として録音）
  4. ⏹️ 電話終了後「録音停止」をタップ
  5. 💾 「履歴に保存」でデータを保存

📚 用語辞書機能の活用

  • リアルタイム変換: 音声認識中に自動で言い間違いを修正
  • デフォルト辞書: IT・AI・KPIなどのビジネス用語28個を内蔵
  • カスタム追加: 会社固有の用語や人名を追加可能
  • 手動適用: 既存の文字起こしにも辞書を適用可能

🤖 Claude連携で要約作成

  • 5種類のテンプレート: 汎用・会議・営業・サポート・インタビュー
  • カスタムテンプレート: 独自の要約形式を作成・保存
  • ワンクリック連携: プロンプトをコピーしてClaude.aiを自動オープン

❓ よくある質問＆トラブル解決

🎤 マイクが許可されない・音声が認識されない ▼

Chrome/Edgeの場合:

  1. アドレスバーの左側にある🔒アイコンをクリック
  2. 「マイク」の設定を「許可」に変更
  3. ページを再読み込み

Safariの場合:

  1. Safari メニュー > 設定 > Webサイト
  2. 「マイク」を選択
  3. このサイトを「許可」に設定
📚 用語辞書が効かない・変換されない ▼
  1. 「🔄 リアルタイム自動変換」がONになっているか確認
  2. 用語が正しく登録されているか確認
  3. 手動で「📚 辞書適用」ボタンを試す
  4. 「📚 デフォルト辞書」で基本用語を読み込み
⚡ 動作が重い・エラーが頻発する ▼
  1. ブラウザのキャッシュをクリア
  2. 用語辞書で「🎯 最適化」を実行
  3. 不要な履歴を削除
  4. ページを再読み込み
  5. Chromeの最新バージョンを使用
🤖 Claude連携がうまくいかない ▼
  1. 文字起こしが完了しているか確認
  2. 適切なテンプレートを選択
  3. 「📋 プロンプトをコピー」で手動コピー
  4. Claude.aiで新しいチャットを開始
  5. Ctrl+Vでプロンプトを貼り付け
